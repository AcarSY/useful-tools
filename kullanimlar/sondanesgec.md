# sondanesgeç fonksiyonu için örnek kullanım

```js
const useful = require('useful-tools')

const yazı = 'Merhaba bu modül çok güzel!'
const uzunluk = 7
console.log(useful.sondanesgeç(yazı, uzunluk))

// Örnek Çıkış: Merhaba bu modül
```
